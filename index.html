AI-Powered Workplace Assistance System

Graduate Research Assistant | Arizona State University

August 2025 - Present

Project Overview

Developed an advanced computer vision pipeline to create assistive technology for visually impaired workers at a non-profit organization. This system enables real-time monitoring and guidance for workplace tasks, significantly improving accuracy and independence.

Key Achievements

  • 95% Object Tracking Accuracy: Developed computer vision pipeline using YOLO-World and SAM2 to enable real-time video segmentation for assistive technology, achieving exceptional accuracy in object tracking for visually impaired workers
  • 40% Error Reduction: Engineered AI-powered workplace assistance system that tracks box selection and item sorting activities through text-prompted segmentation, dramatically reducing workplace errors and improving task completion confidence for blind employees
  • Real-time Processing: Implemented accessible video analysis solution processing 30+ fps real-time footage to provide audio feedback for package sorting tasks
  • Significant Impact: Directly impacting 50+ visually impaired workers' daily productivity and workplace independence

Smart Cane for Visually Impaired Users

Undergraduate Research Assistant | University of Guelph

May 2025 - August 2025

Project Overview

Developed an intelligent navigation aid for visually impaired individuals by creating a smart cane that combines computer vision, distance sensing, and real-time feedback systems. This project integrates multiple sensor modalities to provide comprehensive obstacle detection and avoidance guidance.

Key Achievements

  • Sensor Fusion System: Engineered a real-time sensor fusion system on Raspberry Pi 5, integrating YOLO-based object detection and ultrasonic distance sensing via ROS2 to identify obstacles and measure proximity accurately
  • Dual-Mode Feedback: Implemented automated feedback system with dual-mode output including audio announcements and haptic vibration patterns based on fused sensor data and object classification confidence thresholds
  • Real-time Decision Pipeline: Integrated deep learning object detection (YOLO) with distance sensing to create rule-based decision pipeline for real-time obstacle avoidance alerts on embedded hardware
  • Embedded AI: Successfully deployed deep learning models on resource-constrained embedded hardware for real-time performance

Technologies Used

  • Raspberry Pi 5
  • ROS2 (Robot Operating System)
  • YOLO (Object Detection)
  • Ultrasonic Distance Sensors
  • Python
  • Audio and Haptic Feedback Systems
